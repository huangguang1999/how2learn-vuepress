# 说明

此仓库用于搭建个人的博客系统，内含大量文章，如需模板，请参考官网和其他示例。



## 安装、启动和打包

在使用前请先安装VuePress
```sh
$ npm install -g vuepress
```

使用`npm run dev`启动项目，启动后在浏览器中使用`localhost:3000`进行访问
```sh
$ npm run dev
```

使用`npm run build`打包项目
```sh
$ npm run build
```
